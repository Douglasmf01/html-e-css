Trabalhando com listas

listas ordenadas

É a lista na qual a ordem faz a diferença. usando a tag ol, que significa ordered list eu crio uma lista que nomeia altomaticamente seus itens independente das minhas alterações. para cada item adicionado é necessário usar a tag li que significa list item:

  1. este
  2. é
  3. exemplo
  4. de
  5. lista
  6. ordenada

usando tambem type depois de ol posso usar letras ao inves de nnumeros e até números romanos

  1. este
  2. é
  3. exemplo
  4. de
  5. lista
  6. ordenada

listas não ordenadas

Já na lista sem ordem, eu uso o ul, que significa unordered list, e tambem a tag li

  • este
  • é
  • um
  • exeplo
  • de
  • lista
  • sem
  • ordem

usando type posso colocar uma bolinha vazia e até quadrado

  • desse
  • jeito
  • aqui
